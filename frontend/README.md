# NeoCareSync Frontend

React TypeScript frontend for NeoCareSync - High Risk Pregnancy Monitoring System

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
# Create .env file
VITE_API_URL=http://localhost:3000
```

3. Run development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Patient Dashboard
- Doctor Dashboard
- Admin Dashboard
- Risk Prediction (manual + document upload)
- Appointment Management
- Risk Trend Visualization

## Tech Stack

- React 18
- TypeScript
- Vite
- React Query
- React Router
- Tailwind CSS
- Recharts
